\section{Introduction}
\label{sec:introduction}

The rapid evolution of software development paradigms necessitates robust and intelligent security auditing solutions that can keep pace with increasingly complex codebases and sophisticated attack vectors. CodeSentinel addresses this critical need by integrating advanced AI techniques, specifically local Large Language Models (LLMs), with secure, containerized execution environments to provide a comprehensive and privacy-preserving security review platform \cite{bouzid2025}. 

This approach facilitates deep semantic analysis of code for vulnerability detection, moving beyond traditional signature-based methods to identify subtle logic flaws and architectural weaknesses \cite{ferrag2025, yao2024}. Unlike conventional static or dynamic analysis tools, CodeSentinel leverages LLMs to analyze code structure, detect intricate patterns, and propose remediation strategies, thereby enhancing the efficiency and accuracy of vulnerability identification \cite{sheng2025}. 

Furthermore, the platform's sandboxed dynamic execution capability within Docker containers allows for the safe emulation of identified exploits, providing empirical validation of potential vulnerabilities without compromising the integrity of the host system \cite{moia2026}. This dual-pronged strategy---combining advanced semantic intelligence with secure, isolated dynamic testing---enables a more thorough and reliable assessment of software security posture \cite{omar2023}.
